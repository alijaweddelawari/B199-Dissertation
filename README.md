An Analysis of Artificial Intelligence Representation in Afghan News Media Using Natural Language Processing

Overview

This repository contains the data and code supporting the undergraduate dissertation analysing how Artificial Intelligence (AI) is represented in Afghan news media using Natural Language Processing (NLP).

The project applies sentiment analysis (VADER) and topic modelling (LDA) to a manually constructed dataset.Afghan AI-related news articles published between 2018 and 2025.

The repository is provided for transparency, reproducibility, and examiner verification.

Contents
	•	Jupyter Notebook containing the full analysis pipeline
	•	Final Afghan AI news dataset and labelled outputs
	•	External validation dataset (Reuters headlines)
	•	Supplementary materials referenced in the dissertation

Methodology Summary
	•	Text preprocessing using Python and NLTK
	•	Sentiment analysis using VADER (Positive / Neutral / Negative)
	•	External validation using a labelled Reuters dataset
	•	Topic modelling using Latent Dirichlet Allocation (LDA)
	•	Joint sentiment–topic interpretation and visualisation
